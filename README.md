# NextGenTeamBlue

## Using Python 3.13.9

## Create a Virtual Environment
```
  python -m venv venv
```
Then, activate the virtual environment.
Windows
```
  .\venv\Scripts\activate
```

macOS/Linux
```
   source venv/bin/activate
```

## Install pip packages
```
  pip install -r .\src\requirements.txt
```
